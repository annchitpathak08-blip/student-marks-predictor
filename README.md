# student-marks-predictor
Predicts a student's final grade (G3) using Multiple Linear Regression Model on the UCI Student Performance dataset with data preprocessing, one-hot encoding, evaluation metrics, and visualizations made using seaborn

# Features

- Data preprocessing
- One-Hot Encoding
- Multiple Linear Regression
- Model Evaluation
- Prediction for new student data
- Data Visualization

# Dataset

- UCI Student Performance Dataset
- Target Variable: G3 (Final Grade)

# Libraries Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

# Evaluation Metrics

- MAE
- MSE
- RMSE
- R² Score

# Visualizations

- Actual vs Predicted
- Error Distribution
- Feature Importance
